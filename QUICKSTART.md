# 快速开始指南

## 功能说明

本应用已添加以下新功能：

### 1. 下载速度测试
- 点击"测速"按钮可以测试单个IP的下载速度
- 测速结果会显示在"下载速度"列中
- 支持自定义测速URL

### 2. 最低下载速度过滤
- **控制面板过滤**: 在开始扫描前设置"最低下载速度 (MB/s)"
- **结果页过滤**: 在详细测试页面设置过滤速度，点击"过滤"按钮
- 设置为 0 表示不过滤
- 过滤后只显示满足速度要求的IP

### 3. 自定义测速URL
- 在控制面板的"测速 URL"字段可以自定义测速服务器
- 默认: `speed.cloudflare.com/__down?bytes=100000000`
- 设置会自动保存

## 使用流程

1. **设置参数**
   - 选择IP类型（IPv4/IPv6）
   - 设置并发数、端口、延迟阈值
   - （可选）设置最低下载速度
   - （可选）自定义测速URL

2. **开始扫描**
   - 点击"开始扫描与测试"
   - 等待扫描完成

3. **选择数据中心**
   - 在"数据中心汇总"中选择要测试的数据中心
   - 点击"选择测试"

4. **查看结果**
   - 切换到"详细测试"标签页
   - 查看延迟测试结果
   - 点击"测速"按钮测试下载速度

5. **过滤结果**
   - 设置最低下载速度
   - 点击"过滤"按钮
   - 只显示满足速度要求的IP

## 构建APK

### Windows
```cmd
build.bat
```

### Linux/macOS
```bash
chmod +x build.sh
./build.sh
```

详细说明请参考 `BUILD.md` 和 `README.md`。
